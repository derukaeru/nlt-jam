Change worlds to solve platforming puzzles!
This is a 3d puzzle platforming game that utilizes two "worlds" that contain different objects
Was originally made for NLT-JAM with the theme: Parallel Universe

Currently has 3 levels but will be further developed in the future!

Made using Godot

Controls:

WASD - Move
SPACE - Change Worlds
